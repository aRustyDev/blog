# Phase 4: GEO (Generative Engine Optimization)

## Goal

Optimize content structure for AI search engines (Perplexity, Google SGE, Bing Copilot).

## Implementation Steps

1. **Clear headings**: ensure H2/H3 hierarchy is scannable
2. **Concise definitions**: first paragraph of each section answers "what is X"
3. **Key takeaways**: add structured takeaway sections that AI can extract
4. **Citation-friendly**: numbered references, linked sources
5. **Structured data**: schema.org markup enables rich AI-generated answers
6. Content guidelines doc for authors

## Files to Create

- `docs/src/guides/geo-guidelines.md` — authoring guidelines for AI-optimized content

## Acceptance Criteria

- [ ] Guidelines documented
- [ ] Existing posts audited against guidelines
- [ ] Key takeaway sections added to major posts
