## Overview

A deep-dive overview of PCB design covering EDA tools (KiCad, Altium, Eagle), PCB fundamentals (layers, traces, vias, pads, copper pours), component selection (through-hole vs SMD, package types, sourcing), design rules (clearance, trace width, DRC), the manufacturing process (gerber generation, pick-and-place, reflow), and prototyping services (JLCPCB, PCBWay, OSH Park). Fundamentals-first approach: understand the entire PCB design pipeline before diving into specific projects. Entry point for the PCB design series.
