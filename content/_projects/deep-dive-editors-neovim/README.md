# Deep Dive: Neovim

> From Vim fork to editor platform — understanding Neovim's architecture, ecosystem, and the platform-first philosophy that makes it more than just an editor.

## Status

**Phase**: ideation

## Overview

A deep-dive on Neovim covering its architecture (Lua-first configuration, built-in LSP client, Tree-sitter integration, async job control), the plugin ecosystem (lazy.nvim, telescope, nvim-cmp, lsp-zero), configuration philosophy (init.lua vs distributions like LazyVim/NvChad/AstroNvim), terminal integration, DAP support, and Neovim as a complete IDE. Platform-first approach: Neovim isn't just an editor, it's a platform for building your editor.
