# Deep Dive: vi

> From ADM-3A terminals to POSIX standard — understanding the original visual editor and the constraints that shaped modal editing.

## Status

**Phase**: ideation

## Overview

A deep-dive on vi covering its creation by Bill Joy (1976), the ex line editor heritage, the original terminal constraints that shaped modal editing, vi's POSIX standardization, how vi differs from Vim, nvi and other true vi implementations, vi on modern systems, and vi's influence on every modal editor that followed. History-first approach: understanding vi's constraints explains why modal editing works the way it does.
