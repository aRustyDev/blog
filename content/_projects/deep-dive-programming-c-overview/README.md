## Overview

An overview of C programming covering C's design philosophy (trust the programmer, close to the metal), the compilation model (preprocessor, compiler, assembler, linker), memory model (stack, heap, static, manual management), undefined behavior (what it is and why it matters), the standard library, C standards evolution (C89 through C23), toolchain (gcc, clang, make, cmake), and why C remains essential despite modern alternatives. Foundation-first approach: understanding C means understanding the machine.
