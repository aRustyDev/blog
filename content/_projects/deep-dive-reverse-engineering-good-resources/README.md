## Overview

A curated guide to reverse engineering learning resources covering foundational books (Practical Reverse Engineering, Reverse Engineering for Beginners, The IDA Pro Book), online courses (OpenSecurityTraining2, Malware Unicorn, begin.re), CTF platforms (picoCTF, pwnable.kr, crackmes.one), community resources (r/ReverseEngineering, Discord servers, conferences), practice materials (MalwareBazaar, VirusTotal), and a recommended learning path from beginner to intermediate. Path-first approach: the hardest part of learning RE is knowing what to study in what order.

## Notes

- Opinionated sequencing differentiates from typical link-dump resource lists
- Resource categorization by learning style (reading, watching, doing) helps different learners
- Honest assessments of each resource prevent wasted study time
- CTF as deliberate practice framing is more useful than CTF as competition
- Tool-agnostic foundation emphasis prevents the common mistake of learning tools before concepts
