> One pattern, many faces — understanding the data/code confusion behind SQL injection, XSS, command injection, and prompt injection.

## Overview

An ELI5 introduction to injection-based attacks covering the unifying pattern (data/code confusion), the major injection types (SQL, XSS, command, LDAP, SSTI, header, prompt), and the universal defense principle (separate code from data). Analogy-first approach using the mad libs metaphor, with before/after visualizations for each attack type. Accessible to non-technical readers while remaining technically accurate.
