## Overview

A deep-dive exploring Git's content-addressable object store, the directed acyclic graph, refs, the index, packfiles, and how common operations (commit, merge, rebase, cherry-pick) manipulate these structures. Readers move from "I memorized the commands" to "I understand the model." Includes hands-on exploration with `git cat-file` and DAG evolution diagrams.
