# Deep Dive: ed

> From teletypes to every text tool — understanding the standard editor and the command language that spawned grep, sed, vi, and regex itself.

## Status

## Overview

A deep-dive on ed covering its creation (Ken Thompson, 1969), why a line editor made sense on teletypes, ed's command language (the ancestor of sed, grep, and vi's ex mode), regular expressions in ed (where regex was born for editing), ed in the modern world, ed scripting, the famous "?" error message, and ed's profound influence on every text editor and text processing tool that followed. Origins-first approach: every text editor descends from ed, and understanding ed illuminates the entire lineage.
