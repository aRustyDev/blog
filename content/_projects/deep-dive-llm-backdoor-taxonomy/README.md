# Deep Dive: LLM Backdoor Taxonomy

> A security practitioner's framework for understanding, detecting, and defending against LLM backdoor attacks.

## Status

**Phase**: ideation
**Persona**: not set

## Overview

This project presents a practitioner-oriented taxonomy of LLM backdoor attacks — maliciously embedded behaviors triggered by specific inputs that are distinct from both prompt injection and jailbreaking. Using a three-axis classification (trigger type x injection method x impact), the post organizes fragmented ML security research into an actionable framework for threat modeling. It maps attack surfaces across the full LLM pipeline (pre-training through deployment), surveys notable research and real-world incidents, provides a detection effectiveness matrix with honest assessments of current capabilities, and delivers actionable recommendations segmented by role (security teams, ML teams, organizations).
