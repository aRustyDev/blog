## Overview

A deep-dive on Yazi covering its architecture (async I/O, Rust-based), integration with Neovim (yazi.nvim plugin), file previewing (images in terminal, syntax highlighting, PDF/video previews), bulk operations, comparison to other file managers (ranger, lf, nnn, broot), and how Yazi complements a Neovim-based development workflow. Integration-first approach: Yazi shines when it's part of a larger terminal workflow, not as a standalone tool.
