## Overview

A research-driven deep-dive on LLM observability for agent development. Covers the seven observable layers of the AI stack, four metric domains (performance, cost, quality, reliability), agent-specific observability patterns, open-source tooling (OpenLLMetry, LangSmith, Langfuse, Arize Phoenix), evaluation frameworks (RAGAS, DeepEval), token optimization, hallucination detection, and OpenTelemetry GenAI standards. Instrumentation-first approach: observability from day zero, not after production incidents.
