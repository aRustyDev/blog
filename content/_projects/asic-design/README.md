## Overview

A multi-post project covering the full custom ASIC design pipeline using open-source tools (OpenLane 2, Yosys, Magic, SkyWater 130nm PDK) and open shuttles (Efabless). Dual structure: "Zero to ASIC" educational series explaining each stage, plus "My First ASIC" narrative documenting the personal experience. Targets software engineers curious about hardware design.
